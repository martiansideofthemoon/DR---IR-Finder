Department Rank / Institute Rank Finder

This is a simple Django based UI which can help you find you department rank or your institute rank based on data entered by students.
How to use this project :-
It is a Django based project so it runs in the standard way, by typing "python manage.py runserver" on a terminal window which has been navigated to the project directory. Log in to "/drirfinder" to open the Django form where you can add your Roll Number, CPI, Department. Press the display button to see all the people sorted in rank order.
